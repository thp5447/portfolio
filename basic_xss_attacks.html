Project Overview

This code demonstrates a malicious script designed to steal cookies by sending the user's document.cookie to a remote webhook URL using fetch requests. It employs obfuscation techniques, such as converting strings into character codes and manipulating strings dynamically, to evade detection. This script is an example of a Cross-Site Scripting (XSS) attack and highlights the importance of securing web applications against such vulnerabilities.

Key Features

  • Steals cookies using malicious scripts
  • Employs obfuscation techniques to evade detection
  • Highlights security vulnerabilities in web applications

Technologies Used

  • JavaScript
  • Web Application Security
  • Obfuscation Techniques

Challenges Faced

The main challenge was understanding and demonstrating obfuscation techniques effectively without undermining ethical boundaries. The project underlines the significance of robust security measures against such attacks.
